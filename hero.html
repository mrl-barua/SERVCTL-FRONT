Self-host in minutes.

One command. Your infrastructure, your rules.

install.sh 
# Clone the repository
git clone https://github.com/mrl-barua/SERVCTL-FRONT.git
cd SERVCTL-FRONT
# Install dependencies
npm install
# Configure environment
cp .env.example .env
# Start the dev server
npm run dev
